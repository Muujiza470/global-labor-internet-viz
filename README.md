# Global Labor & Internet Access Data Visualization (2000–2023)

## Overview
This project integrates multi-source global datasets to analyze relationships between internet access, migration, and labor participation (2000–2023).  

The goal was to explore how digital access correlates with foreign-born labor participation across different regions.

---

## Data Sources
- OECD
- ILO
- ITU
- United Nations

---

## Tools Used
- Python (data cleaning, merging, feature engineering)
- Tableau (interactive dashboards, maps, animated scatter plots, dual-axis charts)

---

## What I Did
- Integrated and cleaned multi-source datasets using Python
- Created derived features to align country and year-level indicators
- Built an interactive Tableau dashboard
- Designed visual storytelling elements to highlight global patterns
